# 每日使用流程

## 🎯 完整的每日工作流程

### 晚上 22:00 - 收到复盘提醒

飞书自动收到消息：
```
📊 每日复盘

🌙 晚上好！今天的任务完成情况如何？

📋 今日任务清单：

🔄 用户登录功能
   进度：[■■■■■■■■□□] 80%
   象限: Q1

💬 请回复以下内容：
1. 今天完成了哪些任务？进度如何？
2. 明天计划做什么？
3. 有哪些任务需要暂缓？

示例：完成了用户登录功能80%，明天做数据库设计Q2任务
```

### 晚上 22:00-23:00 - 你回复进展

你有两种方式更新任务：

#### 方式1：直接在飞书回复（推荐）

直接在飞书中回复机器人消息：
```
完成了用户登录功能90%，这是Q1任务
明天做数据库设计Q2任务
```

**注意**：飞书机器人是单向的，它看不到你的回复。所以你需要：

1. 复制你的回复内容
2. 前往 GitHub Actions：https://github.com/Zihui1112/ai-email-coach/actions
3. 选择 **"处理用户回复"** workflow
4. 点击 **"Run workflow"**
5. 粘贴你的回复内容
6. 点击运行

#### 方式2：直接在 GitHub Actions 输入

1. 前往：https://github.com/Zihui1112/ai-email-coach/actions
2. 选择 **"处理用户回复"** workflow
3. 点击 **"Run workflow"**
4. 在输入框中输入你的更新：
   ```
   完成了用户登录功能90%，这是Q1任务
   明天做数据库设计Q2任务
   ```
5. 点击运行

### 晚上 23:00 - 收到跟进提醒

飞书自动收到消息：
```
📊 每日跟进

🌙 晚安！今天的任务都完成了吗？

如果有任务更新，请：

📝 方式1：直接在这里回复（推荐）
例如：完成了用户登录功能90%，这是Q1任务

🔗 方式2：前往 GitHub Actions
https://github.com/Zihui1112/ai-email-coach/actions
选择 '处理用户回复' → Run workflow

💤 如果今天没有更新，就好好休息吧！
明天继续加油！
```

这是一个友好的提醒，如果你还没更新任务，可以在这时候更新。

### 更新后 - 收到 AI 反馈

当你通过 GitHub Actions 提交回复后，飞书会收到 AI 的反馈：
```
📊 任务更新反馈

✅ 用户登录功能
   进度：[■■■■■■■■■□] 90%
   象限: Q1
   状态: 进行中

🆕 数据库设计
   进度：[□□□□□□□□□□] 0%
   象限: Q2
   状态: 进行中

💪 继续加油！
```

## 💡 使用技巧

### 1. 快速更新
如果你很忙，可以简单回复：
```
用户登录90%完成
```

AI 会自动识别任务名称和进度。

### 2. 批量更新
可以一次更新多个任务：
```
完成了用户登录功能90%，这是Q1任务
完成了数据库设计50%，这是Q2任务
暂缓支付功能，这是Q3任务
```

### 3. 如果忘记更新
没关系！第二天的复盘提醒会显示昨天的进度，你可以在那时候更新。

### 4. 周末不想被打扰
周末也会收到复盘提醒，但你可以选择不回复。系统会继续正常运行。
