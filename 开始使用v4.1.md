# 🚀 开始使用 v4.1

## ⚡ 3步快速部署

### 第1步：清理数据库（3分钟）
1. 打开 https://supabase.com/dashboard
2. 进入 SQL Editor
3. 复制粘贴 `clear_tasks_data.sql` 并执行

### 第2步：推送代码（2分钟）
```bash
git add .
git commit -m "v4.1: Complete UI optimization"
git push origin main
```

### 第3步：测试（5分钟）
1. GitHub Actions → 手动触发 "每日复盘提醒"
2. 检查邮件格式
3. 完成！
