# 解决Git推送问题

## 当前状态

✅ 远程仓库地址已正确设置：`https://github.com/Zihui1112/ai-email-coach.git`

## 推送步骤

### 方式1：使用GitHub Desktop（最简单）

1. 下载并安装GitHub Desktop：https://desktop.github.com
2. 打开GitHub Desktop
3. File → Add Local Repository → 选择 `D:\Plan_maker`
4. 点击 "Publish repository"
5. 选择你的账号 Zihui1112
6. 仓库名：ai-email-coach
7. 点击 "Publish repository"
8. 完成！

### 方式2：使用Personal Access Token

#### 步骤1：生成Token

1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token" → "Generate new token (classic)"
3. Note：填写 `AI Email Coach`
4. Expiration：选择 `No expiration`（永不过期）
5. 勾选权限：
   - ✅ repo（所有子选项）
6. 点击 "Generate token"
7. **重要**：复制生成的token（类似：`ghp_xxxxxxxxxxxx`）

#### 步骤2：推送代码

在命令行执行：

```bash
git push -u origin main
```

当提示输入用户名和密码时：
- Username: `Zihui1112`
- Password: 粘贴刚才复制的token（不是GitHub密码！）

### 方式3：配置Git凭据

如果不想每次都输入token，可以保存凭据：

```bash
# 保存凭据
git config --global credential.helper store

# 然后推送
git push -u origin main
```

输入一次token后，以后就不用再输入了。

## 验证推送成功

## 如果还是失败

### 检查1：仓库是否存在

访问 https://github.com/Zihui1112/ai-email-coach

如果显示404，说明仓库不存在，需要先创建：
1. 访问 https://github.com/new
2. Repository name: `ai-email-coach`
3. 选择 Public
4. **不要**勾选任何初始化选项
5. 点击 "Create repository"

### 检查2：网络问题

如果推送很慢或超时，可能是网络问题。可以：
1. 使用VPN
2. 或者使用GitHub Desktop（更稳定）

### 检查3：权限问题

确保你已登录GitHub账号 Zihui1112，并且有权限推送到该仓库。
