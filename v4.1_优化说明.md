**功能**:

## 📝 用户使用指南

### 回复格式（简化版）

**基本格式**:
```
Q1: 1完成; 2进度50%; 3暂缓
Q2: 1进度80%
新增：写论文 Q1
暂缓任务1恢复到Q1
```

**规则**:
1. 用象限+编号引用任务（Q1: 1, 2, 3）
2. 多个操作用分号分隔
3. 新增任务格式：`新增：任务名 象限`
4. 恢复暂缓任务：`暂缓任务X恢复到QY`

### 邮件格式（极简版）

**任务清单**:
- 按象限分组
- 每个任务一行：编号 + 名称 + 进度条
- 象限之间用空行分隔
- 无多余分隔线

**反馈邮件**:
- 显示完整任务名称
- 清晰的操作结果
- 经验值和金币奖励

## ⚠️ 注意事项

1. **数据清理不可逆**：执行 `clear_tasks_data.sql` 会永久删除所有任务数据
2. **游戏化数据保留**：等级、经验值、金币不会被删除
3. **今晚重新开始**：清理后从今晚的每日复盘开始添加新任务
4. **回复格式兼容**：新旧格式都支持，但推荐使用简化格式
